# Dedication

This project is dedicated to builders who want powerful tooling with clear, practical documentation.

The goal of this SDK and docs set is simple:
- Make first-time usage fast and low-stress
- Keep advanced paths possible without guesswork
- Document real behavior, not idealized behavior

`codex-sdk-unofficial` stands on top of the Codex CLI and exists to help the community ship quickly, safely, and confidently.

> This project is community-maintained and is not an official OpenAI SDK.
